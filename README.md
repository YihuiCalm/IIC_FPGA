# IIC_FPGA
IIC implementation using Verilog on Vivado, based on ARTY-7 35T FPGA and AT24C02 EEPROM.

Chipscope Waveform below shows writting 0x45 to memory address 0x23 of a EEPROM with device address of 0b1010000:
